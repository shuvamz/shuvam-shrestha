Types of Selectors

Type Selectors: HTML Element names are refered to as type selectors and they match all occurances of the particular element. section, footer, p, body, etc. are examples of type selctors.

Class Selectors: Class can be applied to many occurances of a particular element. We say the element belongs to that class. This is know as a class selector.

ID Selectors: For example, the section id="combine_selector" element can be uniquely selected as it's id has a uniques value, combine_selector. This is known as an id selector. This unique value, combine_selector can be used as an address in a hyperlink, or a selctor for CSS styling or applying Javascript behaviour.

Psuedo Selectors: Pseudo selector is a keyword added to selectors that specifies a special state of the element to be selected. For example :hover will apply a style when the user hovers over the element specified by the selector.

Selector Combination: The type of selector which directs to a specific selector within another selector is known as a Selector Combination. For example, body>p or body.p in CSS will affect all the p tag within body rather than the entire body
