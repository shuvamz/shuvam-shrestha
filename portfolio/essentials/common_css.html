a. What is a markup?
Markup languages are designed for the processing, definition and presentation of text. The language specifies code for formatting, both the layout and style, within a text file

b. What is hypertext and hypermedia?
Hypertext is a text which contains links to other texts. HyperMedia is not constrained to be text-based. It can include other media, e.g., graphics, images, and especially the continuous media - sound and video

c. What is a doctype and why is it important to include it in every HTML document?
document type declaration, or DOCTYPE, is an instruction that associates a particular HTML page with a document type definition

d. What language can be used to style web documents?
CSS (Cascading Style Sheet) can be used to style web documents

e. Which elements are required for a valid XHTML5 document?
Declaration of DOCTYPE in the document is required for validation

f. What is a self closing HTML element? Give an example?
HTML tags that can close within one tag i.e. does not have a seperate closing tag. Example, &

g. What is XML and why is it important to web developers?
a metalanguage which allows users to define their own customized markup languages, especially in order to display documents on the Internet.

h. Describe 4 XML rules?

  • All XML must have a root element.
  • All tags must be closed.
  • All tags must be properly nested.
  • Tag names have strict limits.

i. What is the difference bwtween a "well-formed" and a valid web document?
Valid XML has a DTD associated with it and has been verified against all the rules contained in the DTD in addition to being well-formed. Merely well-formed XML, on the other hand, is not necessarily valid, although it may be.
